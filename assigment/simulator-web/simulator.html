LAYER 7

Application Layer

TCP/IP: Application
🖥️ What it does: The Application layer is where users interact. It handles high-level protocols (HTTP, FTP, DNS). This is what you see — the raw message.

TCP/IP: Transport

LAYER 3

Network Layer

TCP/IP: Internet
📦 What it does: The Network layer wraps each segment into a packet with source and destination IP addresses. Routers operate at this layer to forward packets across networks.
LAYER 2

Data Link Layer

TCP/IP: Network Access
🔗 What it does: The Data Link layer wraps packets into frames with a MAC header and a CRC checksum for error detection. The CRC is calculated as data.length % 2.
LAYER 1

Physical Layer

TCP/IP: Network Access
⚡ What it does: The Physical layer converts all data into raw binary bits (0s and 1s) and transmits them over the wire as electrical signals, light pulses, or radio waves.
TRANSMISSION

Network Transmission

🌐 What it does: Packets travel across the physical network through routers. Packets can be lost (20% chance) or corrupted (30% chance). TCP handles retransmission automatically.
